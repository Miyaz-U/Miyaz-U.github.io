# Miyaz U — Portfolio Website

A personal portfolio website showcasing my education, work experience, projects, skills, and certifications. Built with vanilla HTML, CSS, and JavaScript, and deployed via GitHub Pages.

🔗 **Live site:** [miyaz-u.github.io](https://miyaz-u.github.io/)

## About

This site is a single-page portfolio for **Miyaz U**, an aspiring AI/ML & Python Full Stack Developer and 2025 CSE graduate from Agni College of Technology, Chennai. It presents a quick overview of my background and links out to project demos and contact channels.

## Sections

- **Home** — intro / hero banner
- **About** — short professional summary
- **Education** — academic background
- **Experience** — internships and work history
- **Projects** — featured projects with links to live demos where available
- **Skills** — technical skill tags
- **Certifications** — completed courses and certifications
- **Languages** — spoken languages
- **Achievements** — notable achievements and activities
- **Contact** — email, phone, location, LinkedIn, and GitHub

## Tech Stack

- **HTML5** — page structure
- **CSS3** — styling, layout, and responsive design
- **JavaScript (Vanilla)** — smooth scrolling, scroll-based section reveal animations, mobile nav toggle, and navbar scroll effect
- **[Vanta.js](https://www.vantajs.com/) + [Three.js](https://threejs.org/)** — animated `NET` background effect (loaded via CDN)

## Project Structure

```
.
├── index.html      # Page markup and content

└── README.md
```

## Running Locally

No build step or dependencies are required. Simply clone the repo and open `index.html` in a browser:

```bash
git clone https://github.com/miyaz-u/miyaz-u.github.io.git
cd miyaz-u.github.io
open index.html   # or double-click the file
```

For live-reloading during development, you can also serve it with any static server, e.g.:

```bash
npx serve .
```

## Deployment

The site is deployed via **GitHub Pages** directly from this repository.
